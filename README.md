# E-commerce Customer Segmentation

## Overview
This project analyzes e-commerce customer data to identify different customer segments and predict high-value customers.

## Methods
- Data cleaning using pandas
- RFM (Recency, Frequency, Monetary) analysis
- Customer segmentation using KMeans
- Visualization using matplotlib and seaborn
- High-value customer prediction using Random Forest

## Results
- Customers are grouped into high-value, medium-value, and low-value segments
- High-value customers have higher spending and purchase frequency
- Recency helps identify potential churn customers

## Business Insights
- High-value customers: focus on retention strategies
- Medium-value customers: targeted promotions
- Low-value customers: re-engagement campaigns

## Tools
Python, Pandas, Scikit-learn, Matplotlib
